# 📒 DollDollNote

> 一个用 Godot 制作的**白板式手账 / 笔记**桌面应用——在可缩放、可平移的纸张上，自由放置文本、图片、形状、涂鸦、emoji 贴纸，配上 Q 弹动效和治愈的画布特效，用来记手账、写日记、做备忘。

私人项目，**Q 弹、可爱、美观**优先。

---

## ✨ 功能亮点

- **🎛 白板式编辑**：可缩放（`Ctrl+滚轮`）、可平移（中键拖拽）的纸张画布
- **🧩 多种 block**：文本（BBCode + Markdown）、图片、形状、手绘涂鸦、emoji / 颜文字贴纸
- **✋ 自由变换**：拖拽移动、边缘缩放、`Ctrl+左键`旋转、调层级 / 透明度 / 旋转
- **📦 多选 & 框选**：空白拖拽框选多块，批量移动 / 删除 / 复制
- **🪀 Q 弹动效**：spawn 弹出、删除消失、拖拽倾斜、选中呼吸、闲置微浮动
- **🌌 画布粒子**：鼠标星光、点击迸发、流星雨、樱花、雨雪、萤火虫、水面涟漪
- **🎬 画面氛围（Shader）**：暗角、复古扫描线、胶片颗粒、暖色漏光
- **🎨 主题配色**：莫兰迪 / 马卡龙 / 和纸 / 牛皮纸 / 薄荷 等一键换肤
- **💮 印章工具**：选个图案盖戳，Q 弹砸落 + 回弹动画
- **📅 日记助手**：天气 / 心情 / 时间戳 / 模板 / 运势
- **🗂 模板系统**：内置精致套件 + 自定义模板
- **📝 Markdown 编辑器**：全屏 MD 编辑，一键插回画布
- **↩️ 撤销重做**、**💾 自动保存**、**📓 多笔记管理**、**📤 导出 PNG / JSON**

## 🎮 如何运行

1. 安装 [Godot 4.6](https://godotengine.org/download/)（标准版即可，无需 .NET 版）
2. 启动 Godot → **导入** → 选择本项目的 `project.godot` 文件
3. 打开后按 **F5** 运行（首次会提示选主场景，选 `Main`）

> 数据（笔记、图片、设置）保存在系统的 `user://data/` 目录下，不随项目走。

## 🛠 技术栈

- **引擎**：Godot 4.6 · GDScript 2.0（严格类型）
- **渲染**：Forward+（Windows D3D12）
- **架构**：`BaseBlock` 基类 + 5 个子类（Text / Image / Shape / Draw / EmojiSticker），交互与 Q 弹动画集中在基类
- **数据**：Godot `Resource` 序列化为 `.tres`，保存到 `user://data/`
- **特效**：粒子用手写对象池 + `_draw`；画面氛围用 `canvas_item` shader 叠加
